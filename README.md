# Connect an Analog Joystick to the Pico W
GND to GND (physical pin 38)
+5V to 3V3 Out (physical pin 37)
VRx to GP28 / ADC2 (physical pin34)
VRx to GP27 / ADC1 (physical pin32)
SW to GP18 (physical pin24)
