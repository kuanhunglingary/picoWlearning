# Connect an Analog Joystick to the Pico W
GND to GND (physical pin 38) <br>
+5V to 3V3 Out (physical pin 37) <br>
VRx to GP28 / ADC2 (physical pin34) <br>
VRx to GP27 / ADC1 (physical pin32) <br>
SW to GP18 (physical pin24)
